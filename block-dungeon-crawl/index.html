Block Dungeon Crawl

Explore a procedurally generated dungeon, break crates, collect gems, defeat the Gatekeeper, then find the exit stairs.

Joypad Left stick / D-pad: move A / X / RT: attack B: dash Start: pause
Keyboard WASD / arrows: move Space / mouse: attack Shift: dash Esc / P: pause
Joypad: press any controller button
Tip: open via localhost for the most reliable joypad support.
